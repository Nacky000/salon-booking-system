## app.py

* LINE Messaging APIのWebhook受信
* ユーザーからのメッセージ取得
* ai_parser.py の呼び出し
* spreadsheet.py の呼び出し
* ユーザーへの返信
* 修正メッセージの受付

---

## ai_parser.py

* OpenAI APIとの通信
* 自由記述から日付・時間を抽出
* JSON形式への変換
* 修正内容の解釈

---

## spreadsheet.py

* Google Sheets APIとの通信
* シフトデータの書き込み
* 既存データの更新

---

## scheduler.py

* 定期実行処理
* 未提出者へのリマインダー
* 自動通知

---

## データ

### shift_data

* 日付
* 開始時間
* 終了時間
* メモ

### user_data

* LINE User ID
* 名前
* 提出状況

---

# 拡張機能

※ 実装時に追加

```text
ShiftBot
│
├─ Notification
│    └─ notifier.py
│
├─ Emergency
│    └─ emergency.py
│
├─ Recruitment
│    └─ recruitment.py
│
├─ Reminder
│    └─ reminder.py
│
└─ AmbiguousInput
     └─ confidence_checker.py
```

## notifier.py

* LINE一斉送信
* 不足枠募集

## emergency.py

* 遅刻連絡受付
* 欠勤連絡受付
* 店長への通知

## reminder.py

* 未提出者の抽出
* 自動催促

## confidence_checker.py

* AI解釈結果の信頼度判定
* 曖昧入力検知
* 追加質問生成
