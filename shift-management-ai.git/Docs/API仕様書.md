# API仕様書

## parse_shift()

役割：自由記述を構造化データへ変換する．

入力

"6/10 18-22"

出力

{
"date":"2026-06-10",
"start_time":"18:00",
"end_time":"22:00",
"memo":""
}

---

## write_shift()

役割：Googleスプレッドシートへ書き込む．

入力

{
"name":"植木",
"date":"2026-06-10",
"start_time":"18:00",
"end_time":"22:00"
}

---

## update_shift()

役割：既存データを修正する．

入力

{
"date":"2026-06-10",
"start_time":"20:00",
"end_time":"22:00"
}

---

## send_reply()

役割：LINEへ解析結果を通知する．

例

「6月10日18:00～22:00で登録しました．
修正がある場合はそのまま送信してください．」

---

## remind()

役割：未提出者へ催促メッセージを送る．

送信例

「シフト未提出の方にお送りしています，
シフトの提出期限は本日です．
期限内に送信をお願いします．」
