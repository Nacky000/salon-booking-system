# データ形式

## シフトデータ

```json
{
    "type": "shift",
    "user_id": "",
    "name": "植木",
    "date": "2026-06-10",
    "start_time": "18:00",
    "end_time": "22:00",
    "memo": ""
}
```

---

## AI入力例

入力

```
6/10 18-22
```

出力

```json
{
    "type": "shift",
    "date": "2026-06-10",
    "start_time": "18:00",
    "end_time": "22:00",
    "memo": ""
}
```

---

## 修正入力例

入力

```
10日は20時からです
```

出力

```json
{
    "type": "update",
    "date": "2026-06-10",
    "start_time": "20:00"
}
```

---

## 自由記述メモ

入力

```
10日は19時以降ならいつでも大丈夫です
```

出力

```json
{
    "type": "shift",
    "date": "2026-06-10",
    "start_time": "19:00",
    "end_time": null,
    "memo": "19時以降ならいつでも"
}
```

---

## 遅刻連絡

入力

```
15分ほど遅れます
```

出力

```json
{
    "type": "late",
    "reason": "",
    "minutes": 15
}
```

---

## 欠勤連絡

入力

```
体調不良のため休みます
```

出力

```json
{
    "type": "absence",
    "reason": "体調不良"
}
```

---

## AIが判定できなかった場合

入力

```
来週金曜遅めから
```

出力

```json
{
    "type": "unknown",
    "message": "開始時間が特定できません"
}
```
