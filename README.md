# Salon Booking System

美容院向けWeb予約システム．
インクリメント開発を用いて実装．

## 機能

### インクリメント1
- 予約登録
- 予約一覧表示
- 予約キャンセル
- 空き時間確認

### インクリメント2
- 会員登録
- ログイン・ログアウト

### インクリメント3
- 管理画面
- 予約管理
- メニュー管理
- カレンダー表示

### インクリメント4
- 美容師指定

### インクリメント5
- 予約履歴

## 開発環境
- Python3
- Flask
- JSON
